
# Pigeon Chat

PigeonChat is a fun messaging app for my learning in Java.


## Usage:
Firstly get the repository by pasting this command into the command line:

    git clone https://github.com/filip-skwarczewski/PigeonChat.git

Secondly move into the newly cloned folder by:

    cd PigeonChat

Thirdly run the Main java file.

That's it!

Happy usage!
